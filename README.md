# Forensic-Bpa-Calc-V3
A terminal tool that calculates Bloodstain Pattern Analysis metrics for forensic aid. You pass a path to a bloodstain image in your desktop or directly move the image inside the terminal. It uses Opencv to read the image and calculate length, width and impact angle. It then ask you to enter XY coordinates to calc 2D area of convergence and 3D area of origin. 
Whats new - Automatic OpenCV image scanning for lenght width and impact angle calculation.
            Added 2D area of convergence and 3D origin of height to understand and analyse the bloodstain pattern more              precisely.
